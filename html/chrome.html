Trucs et Astuces : Google Chrome

Couper le son d’un onglet

Cela vous est surement déjà arrivé : en surfant sur internet, vous avez de nombreux onglets ouverts simultanément, et vous tombez sur un site qui lance une vidéo automatiquement, ce qui peut vite devenir ennuyant pour peu que vous soyez en train d’écouter de la musique. Une solution simple permet de mettre certains onglets sur « mute ». Pour cela, il suffit de faire un clic droit sur l’onglet concerné, et de cliquer sur « Couper le son de l’onglet ».

Booster la rapidité de Chrome

Google Chrome est réputé pour ces options de personnalisation avancées. Les utilisateurs ont la possibilité d’essayer des options avant qu’elles ne soient officiellement implémentées au navigateur. Pour cela, tapez « chrome://flags/ » dans la barre de navigation. Vous allez arriver sur la page des options en phase de bêta, Google vous avertissant qu’il s’agit de fonctionnalités en phase expérimentale, il est donc possible que certains plantages apparaissent, vous voilà prévenu.

Beaucoup d’options s’adressent aux développeurs, mais quelques-unes permettent d’améliorer grandement la vitesse de navigation. Elles sont au nombre de trois : « GPU Accelerated Canvas 2D » et « GPU Accelerated Compositing » vont activer l’accélération matérielle tandis que « Web Page Prerendering » va précharger les pages internet.

Le mode multi-utilisateurs

Particulièrement utile si vous êtes plusieurs à utiliser Chrome sur un appareil, ce mode va personnaliser l’expérience de chacun. Pour cela, il faut cliquer sur l’icône de l’utilisateur en haut à droite de la page, puis choisir « Gérer les utilisateurs ». Vous pourrez ensuite ajouter et modifier le profil de toutes les personnes utilisant Chrome. Les profils prennent en compte les favoris, le thème, l’historique de navigation et les données enregistrées.

Cette astuce pour également s’avérer indispensable pour séparer une utilisation professionnelle et une utilisation personnelle.

Activer le mode Incognito par défaut

Analogue à la « Fenêtre de navigation privée », le mode Incognito propose une interface de navigation qui n’enregistre aucune donnée de navigation sur l’ordinateur : adieu les historiques, les cookies, les mots de passe et identifiants, tous seront supprimés à la fin de la session. Pour accéder par défaut à ce mode, il suffit de faire un clic droit sur l’icône de Chrome de votre bureau et sélectionnez « Propriétés ». Ensuite, ouvrez l’onglet « Raccourci », et allez à la fin du champ « Cible », dans lequel vous rajouterez « -incognito » après chrome.exe.

Validez, Chrome se lancera dorénavant toujours en mode Incognito.

Télécharger un fichier streaming

Une astuce permet de récupérer les fichiers audio et vidéo que Chrome joue en streaming. Ces fichiers sont normalement stockés comme des fichiers temporaires et son difficilement récupérable manuellement, mais une petite astuce permet de mettre la main sur certains contenus.

Lorsque vous êtes sur une page de streaming et que la lecture est en cours, faites un clic droit sur la page et sélectionnez « Inspecter ». Vous verrez dans la colonne de droite un onglet « Network », cliquez sur celui-ci et ensuite sur « Size », ce qui affichera les éléments les plus imposants de la page. Il suffit ensuite d’effectuer un clic droit sur l’élément qui vous semble être le contenu streamé (en général l’élément le plus lourd). Ensuite, cliquez sur « Ouvrir dans un nouvel onglet » pour qu’il vous soit possible d’enregistrer le contenu sur votre disque dur, toujours à l’aide du clic droit.
